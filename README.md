Esse é um Projeto [Next.js](https://nextjs.org/) para consumir a [API C#](https://github.com/49ersRecode/reconnectCsharpAPI)

## Iniciando

- Clone o repositório da [API C#](https://github.com/49ersRecode/reconnectCsharpAPI)
- Execute a API
- Clone este repositório
- Instale as dependências: npm install
- Inicie o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.

## Dependências
- axios (v1.6.5): Um cliente HTTP baseado em promessas para fazer requisições a APIs.
- moment (v2.30.1): Uma biblioteca para análise, validação, manipulação e formatação de datas e horas.
- next (v14.0.1): Um framework baseado em React para construir aplicações web.
- react (v18): Uma biblioteca JavaScript para construir interfaces de usuário.
- react-dom (v18): O ponto de entrada para a manipulação do DOM e renderização do servidor para o React.
